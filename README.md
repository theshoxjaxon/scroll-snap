# Scroll-Snapp

+ 1-step . We need a div classed like container and add styles (
  height: 100vh;
  scroll-snap-type: y mandatory;
  overflow-y: scroll;
) 
+ 2-step . Each container must be a sections and add styles (
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  color: white;
  scroll-snap-align: start;
)
